== Description ==

Elementor page builder addon to display posts in the carousel. Useful for generating post carousel from your blog posts with multiple options.

== Key Features ==

* Multiple customization options and easy to use.
* Fully responsive layout for all devices. 
* Multiple features for blog carousel block.
* Compatible with all themes.
* And much moreâ€¦


== Installation ==

1. Go to Plugins > Add New and search for Elementor Post Carousel Addon , install and activate it.
2. We are good to go. No configuration needed.
3. Post Carousel is added under WPCaps Elements on Elementor page customization panel.


== Frequently Asked Questions ==

= What does this plugin needs to work ? =

This is an addon of Elementor page builder plugin so this plugin works if Elementor is installed and activated.

= Do this plugin works with all themes ? =

Yes, this plugin works with all themes.

= Can I add multiple layouts using this plugin ? =

Yes, it comes with 5 layouts options. Additionally, you can use options available to create extra layouts as well.
